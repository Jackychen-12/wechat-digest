# 公众号推送整理与总结助手

现在支持你要的流程：

- 输入**文章地址**，系统自动抓取正文并入库；
- 输入**公众号名称**，系统尝试自动检索最近文章并批量整理（实验功能）；
- 选中文章后，用自然语言生成摘要。

## 使用

1. 打开网页后，在「自动采集」输入文章地址，点击「抓取该文章」。
2. 或输入公众号名称，点击「抓取最近文章」（受第三方检索策略影响，可能失败）。
3. 在文章库中筛选并选择文章，再输入总结指令。

## 技术说明

- 文章抓取使用 `r.jina.ai/http://<目标地址>` 作为只读代理，以便前端直接获取页面文本。
- 公众号抓取通过第三方搜索页面做实验性检索，并提取 `mp.weixin.qq.com/s?...` 链接后自动抓取。
- 当前摘要仍为本地提取式算法；后续可替换成大模型 API。

## 部署

推荐 Vercel：导入仓库后直接 Deploy（仓库已包含 `vercel.json`）。
